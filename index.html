Attribute: charset="UTF-8" used in <meta> tag

Description:UTF-8 is a super efficient encoding system. It can represent a wide range of characters while still being compatible with ASCII. This makes it a sound choice for use in internationalized software.In short, The HTML5 specification encourages web developers to use the UTF-8 character set, which covers almost all of the characters and symbols in the world! and for that reason we can read pages properly

Attribute: name="viewport" content="width=device-width, initial-scale=1.0" atributes in <meta> tag

Description:Setting the viewport to make your website look good on all devices:\n The viewport is the user'sarea of a web page. It varies with the device - it will be smaller on a mobile phone than on a computer screen.You should include the following "meta tag" element in all your web pages:This gives the browser instructions on how to control the page's dimensions and scaling. The width=device-width part sets the width of the page to follow the screen-width of the device (which will vary depending on the device).The initial-scale=1.0 part sets the initial zoom level when the page is first loaded by the browser.
